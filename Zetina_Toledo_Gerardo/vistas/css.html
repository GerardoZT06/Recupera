¡Bienvenido al aprendizaje de CSS!

Aquí aprenderás sobre las propiedades básicas de CSS y cómo aplicar estilos a tu sitio web.

Para modificar los atributos de alguna etiqueta haremos uso de CSS (Cascading Style Sheets). CSS es un lenguaje de hojas de estilo utilizado para describir la apariencia o presentación de un documento escrito en HTML o XML.


En este caso, se están describiendo los estilos para dos tipos de elementos HTML: "h1" y "p". El elemento "h1" se refiere a los encabezados de nivel 1 en HTML. Los estilos definidos aquí se aplicarán a todos los elementos h1 en la página. Los estilos son los siguientes: background-color: lightblue;: Esto significa que el color de fondo de los elementos h1 será azul claro. color: white;: Esto establece que el color del texto dentro de los elementos h1 será blanco. border: 2px solid black;: Esto significa que los elementos h1 tendrán un borde sólido de color negro con un grosor de 2 píxeles. El elemento "p" se refiere a los párrafos en HTML. Los estilos definidos aquí se aplicarán a todos los elementos de párrafo en la página. Los estilos son los siguientes: background-color: lightgrey;: Esto significa que el color de fondo de los elementos de párrafo será gris claro. color: black;: Esto establece que el color del texto dentro de los elementos de párrafo será negro. border: 1px solid black;: Esto significa que los elementos de párrafo tendrán un borde sólido de color negro con un grosor de 1 píxel. Entonces, si tienes un documento HTML con elementos "h1" y "p", estos estilos se aplicarán a esos elementos según las reglas definidas aquí.
