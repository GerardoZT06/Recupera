Funcion de Imprimir un mensaje con un id

En la sección JavaScript, se define una función llamada saludar(). Esta función, cuando se llama, realiza dos acciones: Crea una variable llamada mensaje y le asigna el string "Hola soy: Gerardo". Utiliza document.getElementById("mensaje") para seleccionar el elemento HTML con el ID "mensaje". Luego utiliza .textContent para cambiar el contenido de texto de este elemento al valor de la variable mensaje, que es "Hola soy: Gerardo". En la sección HTML, hay un botón y un párrafo con el ID "mensaje". Cuando haces clic en el botón, se llama a la función saludar(). Como resultado, el contenido de texto del párrafo cambia a "Hola soy: Gerardo".
